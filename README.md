# Calculator
A Simple Calculator is build using
  1. HTML
  2. CSS
  3. Java Script


The Calculator can perform Operations includes:
  1. Addition(+)
  2. Subtraction(-)
  3. Multiplication(*)
  4. Division(/)
  5. EqualsTo(=)
  6. Point(.)...... This can be operated as a Decimal Point when peroforming and calculating the Decimal Numbers.
